# 📈 股票即時看板

一個簡潔美觀的股票價格監控面板，支援台股、美股等多個市場。

## 功能特色

- 🔍 **即時股價** - 自動更新最新股價與漲跌幅
- 🔄 **手動更新** - 點擊「更新股價」按鈕立即刷新
- ⏱️ **自動更新** - 每 3 分鐘自動更新一次
- 💾 **本地儲存** - 股票列表儲存在瀏覽器中
- 📱 **響應式設計** - 支援手機與電腦瀏覽
- 🎨 **深色主題** - 護眼的暗色介面設計

## 使用方法

1. 在輸入框輸入股票代號
   - 美股: `AAPL`, `TSLA`, `MSFT`
   - 台股: `2330`, `0050`, `2317`
2. 點擊「新增」加入監控列表
3. 點擊「刪除」移除不需要的股票
4. 點擊「🔄 更新股價」手動刷新

## 資料來源

- **台股**: 台灣證交所 API
- **美股**: Yahoo Finance API
