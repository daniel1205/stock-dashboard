# 📈 股票即時看板

一個簡潔美觀的股票價格監控面板，支援台股、美股等多個市場。

## 功能特色

- 🔍 **即時股價** - 自動更新最新股價與漲跌幅
- 💾 **本地儲存** - 股票列表儲存在瀏覽器中
- 📱 **響應式設計** - 支援手機與電腦瀏覽
- ⚡ **自動更新** - 每 30 秒自動刷新資料
- 🎨 **深色主題** - 護眼的暗色介面設計

## 使用方法

1. 在輸入框輸入股票代號
   - 美股: `AAPL`, `TSLA`, `MSFT`
   - 台股: `2330.TW`, `0050.TW`, `2317.TW`
2. 點擊「新增」加入監控列表
3. 點擊「刪除」移除不需要的股票

## 資料來源

使用 Yahoo Finance API 取得即時股票資料
